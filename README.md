# CLAMA

Prosty edytor grafik (piny, posty, stories) + narzędzie do kolaży — w jednym oknie, bez instalacji, wszystko działa w przeglądarce (Chrome/Edge).

**Otwórz: https://kandelucky.github.io/clama/**

- **Edytor** — tekst, linie, kształty, obrazy; szablony, własne fonty, eksport PNG/JPG.
- **Kolaż** (przycisk „Kolaż" w górnym pasku albo [collage.html](collage.html)) — układanie zdjęć w siatce, „Do Clamy" wstawia gotowy kolaż do edytora.

Projekty zapisują się lokalnie w Twojej przeglądarce (localStorage) — nic nie jest wysyłane na żaden serwer.

---

A simple graphics editor (pins, posts, stories) + a photo-collage tool in one window. Runs entirely in the browser (Chrome/Edge), no install, no server — your work is saved locally in your browser.

Made by Lasha + Manoni for [Sallora Studio](https://sallorastudio.com).
